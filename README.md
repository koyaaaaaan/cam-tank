# Implementation

## Prepare
This project need to components below.

* ESP32-CAM
* TANK Kit with Dual Motor
* 2ch Motor Driver 
* Battery x 2 
* Servo Motor x 2 (Optional but... I don't recommend.)
* Cam Servo Mount (Optional but... I don't recommend.)
